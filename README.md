# Web3.storage nodejs upload

## Install

```bash
npm install
```

## links

- [web3.storage](https://web3.storage/)

## work

Create token web ui web3 storage.

```bash
cp .env_example .env
# paste token .env
nano .env
# upload dir
node put-files.js ~/dir
# upload files
node put-files.js ~/dir/*
```
